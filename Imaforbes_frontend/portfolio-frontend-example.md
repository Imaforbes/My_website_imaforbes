# Frontend Example Document

This is a demonstration of how a markdown file works within the `my-portfolio-frontend` repository.

## Usage
Markdown files like this (`.md`) are often used to document the project:
* `README.md` for project overviews.
* `CONTRIBUTING.md` for guidelines on how to push code.
* `CHANGELOG.md` for version history.

This specific file doesn't impact your React code. It's just text!
