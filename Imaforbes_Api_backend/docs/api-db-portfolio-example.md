# Backend Example Document

This is a demonstration of how a markdown file works within the `api_db_portfolio` repository.

## Usage
Markdown files (`.md` extension) render formatted text. They are entirely ignored by the PHP server and do not act as executable code. They are highly useful for documenting API endpoints!

```json
{
  "example_endpoint": "/api/projects.php",
  "method": "GET"
}
```

Since it's just documentation, removing or adding this file has zero impact on how your API functions.
